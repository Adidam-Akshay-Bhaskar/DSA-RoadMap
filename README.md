# 🚀 DSA Roadmap

A premium, high-performance Data Structures and Algorithms tracking application designed to help you master 200+ curated LeetCode questions with a distraction-free monochrome aesthetic.

![Dashboard Preview](https://via.placeholder.com/800x450/0d1117/ffffff?text=DSA+Roadmap+Preview)

## ✨ Features

- **200+ Curated Questions**: Balanced set of problems covering 14 core DSA topics.
- **Difficulty Tracking**: Color-coded badges for Easy, Medium, and Hard problems.
- **Smart Progress Tracking**: Real-time progress bars and completion status syncing.
- **Visual Success State**: Completed cards turn solid green for a clear sense of achievement.
- **Integrated To-Do List**: Manage your study schedule directly within the roadmap.
- **Cloud Sync**: Powered by Supabase to keep your progress safe across devices.
- **Mobile Responsive**: Optimized experience for both desktop and mobile browsers.
- **Physical UI Feedback**: Snappy 80ms transitions and tactile press effects on buttons.

## 🛠 Tech Stack

- **Frontend**: Vite + React
- **Backend / Database**: Supabase
- **Styling**: Vanilla CSS (Custom Monochrome Design System)
- **Deployment**: Ready for Vercel / Netlify

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repository**

   ```bash
   git clone <repository-url>
   cd dsa-project
   ```

2. **Frontend Setup**

   ```bash
   cd frontend
   npm install
   ```

3. **Database Configuration**
   The project uses Supabase. Ensure your `supabaseClient.js` has the correct credentials.
   (Check `database/schema.sql` for table structures if setting up a new instance).

4. **Run Locally**
   ```bash
   npm run dev
   ```

## 📂 Project Structure

- `/frontend`: Main React application source code.
- `/database`: SQL schema for setting up the Supabase database.
- `/README.md`: Project documentation.
